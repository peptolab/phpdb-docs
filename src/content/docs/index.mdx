    <div class="card__body">

        <ul class="list list--checklist">

            <li class="list__item">No unpredictable magic or hidden behaviour</li>
        </ul>
    </div>
</a>

    </div>
    <div class="card__body">
        <p>Secure prepared statements and consistent handling of parameters across all SQL operations.</p>
        <ul class="list list--checklist">
            <li class="list__item">Prevents injection vulnerabilities</li>
            <li class="list__item">Improved performance with statement caching</li>
        </ul>
    </div>
</a>
